## Notes for Stripe Setup

1. **Product Names**: Use exactly "TradeMoq Pro" and "TradeMoq Elite" for consistency
2. **Billing**: Set as recurring monthly subscriptions
3. **Currency**: USD
4. **Description Length**: Stripe allows up to 500 characters for product descriptions
5. **Images**: Consider adding product images/logos if available
6. **Metadata**: Add metadata tags like `plan_type: PRO` and `plan_type: ELITE` for easier webhook handling
